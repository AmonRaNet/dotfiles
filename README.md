- i3 (i3gaps, i3blocks, i3lock)
- terminals and shells (fish, zsh)
- tools
- developer: cpp, python, android
- etc

```
config/install
```

- i3
- i3blocks
- compton
- gnome-terminal
- terminator
- etc

Usage:
```
scripts/install
```
